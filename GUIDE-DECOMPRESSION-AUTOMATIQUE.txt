🚀 GUIDE DÉCOMPRESSION AUTOMATIQUE - DevSoftInstaller
======================================================

🎯 **NOUVELLE FONCTIONNALITÉ AJOUTÉE :**
Décompression automatique des archives après téléchargement

✅ **FORMATS SUPPORTÉS :**
- **ZIP** (.zip) → Décompression native PowerShell
- **7-Zip** (.7z) → Via 7-Zip si installé
- **RAR** (.rar) → Via 7-Zip si installé  
- **TAR.GZ** (.tar.gz) → Via 7-Zip si installé
- **TGZ** (.tgz) → Via 7-Zip si installé

⚙️ **CONFIGURATION :**

1. **Décompression automatique** (activée par défaut)
   - ✅ Activée : Toutes les archives sont décompressées automatiquement
   - ❌ Désactivée : Les archives restent telles quelles

2. **Suppression des archives** (désactivée par défaut)
   - ✅ Activée : Les archives sont supprimées après décompression
   - ❌ Désactivée : Les archives sont conservées

🔧 **FONCTIONNEMENT :**

**Étape 1 : Téléchargement**
- Le fichier est téléchargé normalement
- Statut : "✓ Téléchargé"

**Étape 2 : Détection automatique**
- Le script détecte automatiquement le type d'archive
- Vérifie si la décompression est activée

**Étape 3 : Décompression**
- Création d'un dossier avec le nom de l'archive
- Extraction de tous les fichiers
- Statut : "✓ Téléchargé + Décompressé"

**Étape 4 : Nettoyage (optionnel)**
- Suppression de l'archive si configuré
- Conservation des fichiers extraits

📁 **STRUCTURE DES DOSSIERS :**

**AVANT décompression :**
```
C:\DevInstallers\
├── gradle-8.9-bin.zip
├── python-3.12.5-amd64.exe
└── ...
```

**APRÈS décompression :**
```
C:\DevInstallers\
├── gradle-8.9-bin\          ← Dossier créé automatiquement
│   ├── bin\
│   ├── docs\
│   └── ...
├── gradle-8.9-bin.zip       ← Archive conservée (si configuré)
├── python-3.12.5-amd64.exe
└── ...
```

🎛️ **CONTROLES DANS L'INTERFACE :**

✅ **CheckBox "Décompression automatique"**
- Activé par défaut
- Décompresse automatiquement toutes les archives

✅ **CheckBox "Supprimer archives après décompression"**
- Désactivé par défaut
- Supprime les archives après extraction réussie

📊 **EXEMPLES D'UTILISATION :**

**Scénario 1 : Décompression + Conservation**
- ✅ Décompression automatique : Activée
- ❌ Suppression archives : Désactivée
- **Résultat** : Fichiers extraits + archives conservées

**Scénario 2 : Décompression + Nettoyage**
- ✅ Décompression automatique : Activée  
- ✅ Suppression archives : Activée
- **Résultat** : Fichiers extraits + archives supprimées

**Scénario 3 : Pas de décompression**
- ❌ Décompression automatique : Désactivée
- **Résultat** : Archives téléchargées telles quelles

🔍 **DÉTECTION INTELLIGENTE :**

Le script détecte automatiquement :
- **Extensions d'archives** (.zip, .7z, .rar, .tar.gz, .tgz)
- **Disponibilité de 7-Zip** pour les formats avancés
- **Compatibilité PowerShell** pour les formats ZIP

⚠️ **REQUIS ET LIMITATIONS :**

**Requis :**
- PowerShell 5.1+ (pour Expand-Archive)
- 7-Zip (optionnel, pour formats avancés)

**Limitations :**
- Formats non supportés : .iso, .dmg, .pkg
- Archives corrompues : Gestion d'erreur avec logs
- Espace disque : Double espace temporaire pendant extraction

**Logs de décompression :**
